\# Student Manager 🎓



A comprehensive Windows desktop application built with C# and WPF (.NET 10.0) for managing student records, tracking academic performance, and generating grading reports.



\## Features



\* \*\*Student Management:\*\* Add, edit, and remove student profiles (Name, Class, Birth Date, Age calculation).

\* \*\*Grade Tracking:\*\* Assign grades to specific subjects with issue dates and optional descriptions.

\* \*\*Data Persistence:\*\* \* Save and load data using standard Text (`.txt`) format.

&#x20; \* Serialize and deserialize complete student records (including grades) using JSON.

\* \*\*Statistics Dashboard:\*\* Real-time metrics including total students, average grades, and top-performing students.

\* \*\*Report Generation:\*\* Export individual student grade reports and summaries to text files.

\* \*\*Automated Feedback:\*\* Generates contextual motivational comments based on a student's calculated GPA.



\## Tech Stack



\* \*\*Language:\*\* C# 12.0

\* \*\*Framework:\*\* .NET 10.0, WPF (Windows Presentation Foundation)

\* \*\*Architecture:\*\* Multi-layered approach separating Domain Models, Data Services, and UI components.

\* \*\*Data Format:\*\* JSON (System.Text.Json), TXT
